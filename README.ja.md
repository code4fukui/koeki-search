# koeki-search

日本における公益財団法人に関するデータを検索・可視化するWebアプリケーションです。

## デモ

このプロジェクトは以下の2つの主要なページから構成されています:

1.  **検索インターフェース**: [https://code4fukui.github.io/koeki-search/](https://code4fukui.github.io/koeki-search/)
2.  **データダッシュボード**: 検索ページからアクセス可能な統計マップビュー。

## 機能

### 検索インターフェース
*   **キーワード検索**: キーワードを使用して公益財団法人を検索できます。スペース区切りでAND検索、`|`区切りでOR検索が可能です。
*   **都道府県フィルタ**: 検索結果を特定の都道府県に絞り込めます。
*   **全国対象切替**: 内閣府管轄の全国向け法人を含めるか除外するかを切り替えられます。
*   **詳細表示**: 検索結果の法人をクリックして展開し、住所、代表者、事業概要などの詳細情報を表示できます。
*   **外部リンク**: 法人の公式ホームページや、国税庁法人番号公表サイトのページに直接アクセスできます。

### データダッシュボード
*   **インタラクティブな地図**: 全都道府県における公益財団法人の数を日本カラム地図（TabularMaps Japan）上で可視化します。
*   **統計情報**: 地図上の各都道府県に、法人の総数と10万人あたりの法人数を表示します。
*   **コロプレスマップ表示**: 地図を色分けして法人の密度を示し、地域間の比較を容易にします。
*   **ドリルダウン**: 地図上の都道府県をクリックすると、その地域に所在するすべての法人のリストを即座に表示できます。

## データ
このプロジェクトは、[公益法人オープンデータ](https://github.com/code4fukui/koeki-opendata)プロジェクトによって処理されたオープンデータを利用しています。

元のデータは、[国・都道府県公式公益法人行政総合情報サイト 公益法人information](https://www.koeki-info.go.jp/pictis-info/csa0001!show#prepage2)から取得されています。

# koeki-search 公益財団検索

公益財団の情報を検索・閲覧できるウェブアプリです。内閣府管轄の全国向けの公益財団法人を含む、都道府県別の公益財団法人数を一覧で確認できます。

## デモ
https://code4fukui.github.io/koeki-search/

## 機能
- キーワードによる公益財団の検索
- 都道府県別の公益財団法人数の確認
- 公益財団の詳細情報の表示

## データ・API
本アプリは、[公益法人オープンデータ](https://github.com/code4fukui/koeki-opendata)を利用しています。
